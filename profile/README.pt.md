# Perfil da organização CubeShackles

**Proprietário: CubeShackles (liderado pelo fundador).**

Este repositório contém as predefinições da organização no GitHub (workflows
e modelos).

- Regras de contribuição: https://github.com/CubeShackles/cubeshackles/blob/main/CONTRIBUTING.md
- Autoria e ferramentas: https://github.com/CubeShackles/cubeshackles/blob/main/governance/policies/authorship-and-tooling.md
- Mapa de repositórios: https://github.com/CubeShackles/cubeshackles/blob/main/REPOSITORY_MAP.md
- Taxonomia de etiquetas/marcos do GitHub: https://github.com/CubeShackles/cubeshackles/blob/main/docs/GITHUB_TAXONOMY.md

As ferramentas de programação com IA podem assistir a engenharia. Não são
autoras nem proprietárias da CubeShackles.

Cada PR deve conter etiquetas `type:*`, `layer:*`, `risk:*` e um marco da
plataforma, para que bolsas, fundos e reguladores possam classificar a
alteração em segundos.
